 Loaders! 

					

					

						
 A loader is simply a function that takes some code inn and returns a processed version 


module.exports = source => {
   return source
      .replace(/\<code\>(([\n]|.)+?)\<\/code\>/gi,
         (matches, g1) => {
   const code = g1.replace(//g, '>');
   return `<code>${code}</code>`;
});
